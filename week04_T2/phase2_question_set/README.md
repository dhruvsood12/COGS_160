## Targets met

- **20 questions total** — 5 per tool × 4 tools

## Tool-routing principle

Boolean is for the **canon** (anchor authors, instrument names, controlled vocabulary — verifiable hit counts). Natural-language tools (Elicit / Google AI Scholar / Consensus) are for the **frontier** (gaps, contested claims, cross-domain analogies). The routing rationale is grounded in the Phase-1 reflection (`../../week03_T2_phase1/reflection_phase1.md`).
